# Battleship-Simulator - Project
Console based Batteship simulator made with  C++ ( PF semester project ) 

## Description
This is a console-based Battleship game developed in C++ as a Programming Fundamentals project.
The game supports Player vs Computer mode with manual and random ship placement.

## Features
- Colored console output (Windows)
- Manual & random ship placement
- Player vs Computer gameplay
- Score calculation
- Leaderboard stored using file handling

## How to Run
- Compile using a C++ compiler (Windows)
- Run the executable in Command Prompt

## Controls
- Enter row and column numbers to attack
- Follow on-screen menu options
